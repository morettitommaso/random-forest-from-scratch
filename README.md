# Random Forest from Scratch

Build a complete Random Forest classifier from the ground up, starting with the impurity math behind a single decision tree and ending with a bagged ensemble that votes on predictions. Along the way you'll implement splitting, recursion, bootstrapping, and feature subsampling -- the core ideas behind one of the most reliable classical ML models.

## How to run

```bash
python scaffold.py
```
